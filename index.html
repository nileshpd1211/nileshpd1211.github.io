KSVD-OMP Sparse Representation

Abstract

Sparse representation has attracted much attention from researchers in fields of signal processing, image processing, computer vision, and pattern recognition. Sparse representation also has a good reputation in both theoretical research and practical applications. Many different algorithms have been proposed for sparse representation. The main purpose of this project is to provide an experimental study of applying sparse representation in an iterative manner. We approach this study using two techniques namely orthogonal matching pursuit(OMP) and KSVD Task to be done:
  • To create an overcomplete dictionary by using the K-SVD approach
  • Create a sparse distribution using iterative sparse coding
  • Reconstruct the sparse signal by orthogonal matching pursuit(OMP)
Keyword: Sparse Representation, KSVD, OMP, Thresholding Debaising, dictionary learning

Dictionary Learning

A proper selection for dictionary D is important for sparse representation as well as restoration. A key concept is morphological diversity. Common languages motioned in the last section exhibit different abilities for expression: Fourier is best candidate for vibrations; wavelet is proper for isotropic properties; curvelet is best for piece-wise smooth away from C2 contours. As can be seen, although these representations have their advantages in some kind of problems, no single method currently can represent an image desirably and generally. Hence, dicitonary learning method is chosen according in accordance with the requirement of our research problem.

K-means method encodes each sample using a single clustering center, while K-SVD method sparsely encodes the samples using atoms from the dictionary D . K-SVD is consisted of two steps: sparse coding stage searching for the sparse representation for each sample and dictionary update stage revising the dictionary using SVD decomposition. This process can be interpreted as updating the representation and the dictionary in an alternating fashion.
