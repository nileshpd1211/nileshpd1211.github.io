KSVD-OMP Sparse Representation

Abstract

Sparse representation has attracted much attention from researchers in fields of signal processing, image processing, computer vision, and pattern recognition. Sparse representation also has a good reputation in both theoretical research and practical applications. Many different algorithms have been proposed for sparse representation. The main purpose of this project is to provide an experimental study of applying sparse representation in an iterative manner. We approach this study using two techniques namely orthogonal matching pursuit(OMP) and KSVD Task to be done:
  • To create an overcomplete dictionary by using the K-SVD approach
  • Create a sparse distribution using iterative sparse coding
  • Reconstruct the sparse signal by orthogonal matching pursuit(OMP)
Keyword: Sparse Representation, KSVD, OMP, Thresholding Debaising, dictionary learning
